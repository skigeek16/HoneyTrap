# HoneyTrap - AI-Powered Scam Detection Honeypot

An intelligent honeypot API that detects scam messages and engages scammers in conversation to extract intelligence.

## API Endpoint

POST `/v1/chat`

Headers:
- `x-api-key: secret-key-12345`

## Features
- Pattern-based scam detection
- ML-powered phishing detection
- LLM-generated engagement responses
- Intelligence extraction (UPI, bank accounts, phishing links)
